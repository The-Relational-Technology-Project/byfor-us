# By/For Us

A community portal and directory of neighborhood digital tools built in the Outer Sunset — by us, for us.

## What is this?

By/For Us is a launchpad for the Relational Technology Project's neighborhood tech ecosystem. It showcases the local digital tools and projects that have been built by and for Outer Sunset residents, with status indicators, steward contacts, and a way to suggest new ideas.

## Features

- **Tool directory** — Curated list of community-built digital tools (Outer Sunset Today, Community Supplies, Cozy Corner, and more)
- **Status indicators** — See which tools are live, in development, or in planning
- **Steward contacts** — Find who maintains each tool
- **Suggest an idea** — Propose a new tool for the neighborhood

## Tech stack

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) + [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)

## Part of the Relational Technology Project

Built as part of the [Relational Technology Project](https://relationaltechproject.org) — open-source neighborhood tech made in the Outer Sunset, for neighborhoods everywhere.
